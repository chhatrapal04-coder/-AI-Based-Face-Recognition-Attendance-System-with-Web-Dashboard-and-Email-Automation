# 🎯 FaceTrack Pro – AI Attendance System

## 📌 Overview
FaceTrack Pro is an AI-powered Automated Attendance Monitoring System that uses face recognition to mark attendance in real-time. It combines computer vision, a Flask backend, and a Streamlit dashboard to create a smart and efficient attendance solution.

---

## 🚀 Features
- 🎥 Real-time face detection and recognition
- 🧠 Automated attendance marking
- 🌐 Flask-based backend API
- 📊 Interactive dashboard using Streamlit
- 📧 Email notifications for attendance updates
- 📁 Easy record management

---

## 🛠️ Tech Stack
- Python
- OpenCV
- face_recognition
- Flask
- Streamlit
- SMTP (Email Automation)
